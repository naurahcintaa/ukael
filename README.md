Aplikasi mobile perpustakaan digital interaktif yang memungkinkan pengguna untuk menjelajahi, meminjam, dan mengelola koleksi buku secara digital. Dibangun menggunakan Flutter untuk pengalaman pengguna yang cantik dan responsif.

Aplikasi ini dirancang untuk memberikan pengalaman membaca dan peminjaman buku yang mulus bagi pengguna:

Sistem Otentikasi Pengguna:

Halaman Login dengan validasi input (Nama, Email, Password).

Pengguna dapat masuk untuk mengakses koleksi pribadi mereka.

Koleksi Buku Interaktif:

Halaman Home (Perpustakaan Digital) menampilkan sapaan personal ("Welcome, Naura!").

Fitur Pencarian berdasarkan Judul atau Overview buku.

Filter Kategori: All (Semua), Available (Tersedia), On Loan (Sedang Dipinjam) untuk memudahkan navigasi.

Tampilan buku yang menarik dengan detail sampul, rating bintang, dan informasi penerbit.

Menampilkan daftar buku yang Terakhir Diakses dan Rekomendasi.

Linimasa Peminjaman (Timeline):

Halaman Linimasa untuk melacak riwayat peminjaman dan pengembalian buku.

Menampilkan status buku (Dikembalikan).

Fitur interaktif seperti opsi "Berikan Ulasan" (tergantung implementasi lebih lanjut).

Profil Pengguna & Manajemen Akun:

Halaman Profil yang mempersonalisasi informasi pengguna (Nama, Email, Foto Profil).

Menampilkan bagian "Tentang Saya" untuk deskripsi diri.

Menampilkan Riwayat Peminjaman Saya.

Tombol Log Out untuk keluar dari sesi akun.
